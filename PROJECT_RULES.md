# Project Rules: US Market Analyzer

## Language Policy
- **Application Outputs**: All AI-generated content (Macro Analysis, Stock Summaries, Economic Calendar insights, ETF Flow Matrix) must be displayed in **English only**.
- **Dashboard UI**: Labels, buttons, and status indicators should remain in English to maintain a professional fintech aesthetic.
- **Documentation**: Project-specific logs and internal technical documentation should use English.
- **Communication**: Chat interactions may continue in Korean as per the user's general preference, but all terminal/application data reflects this English-only rule.

## Data Refresh & Polling
- **Macro/Calendar**: Periodic background refresh every 1 hour.
- **Status Polling**: Dashboard status polling interval is set to 15 seconds to optimize performance.
