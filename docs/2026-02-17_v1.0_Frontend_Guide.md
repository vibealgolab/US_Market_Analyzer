# Frontend Design & Coding Guide (v1.0)
**Date:** 2026-02-17
**Author:** VibeAlgoLab

## 1. Design System
We aim for a **"Premium Fintech Dark Theme"**. Use the following keywords when requesting designs from AI.

### A. Color Palette (Tailwind CSS)
- **Backgrounds:** `bg-zinc-950` (Main), `bg-zinc-900/50` (Card), `backdrop-blur-md` (Glass Effect)
- **Accents:** 
    - **Primary:** `text-indigo-500` / `bg-indigo-600` (Brand Color)
    - **Success:** `text-emerald-500` (Up/Positive)
    - **Danger:** `text-rose-500` (Down/Negative)
    - **Text:** `text-zinc-100` (Headings), `text-zinc-400` (Body)

### B. Typography
- **Font:** `Geist` (Next.js Default Font)
- **Styling Rules:**
    - **Dashboard Title:** `text-3xl font-black uppercase tracking-tighter text-transparent bg-clip-text bg-gradient-to-r from-white to-zinc-500`
    - **Section Header:** `text-xs font-black uppercase tracking-widest text-zinc-500`
    - **Card Value:** `text-4xl font-black text-white tracking-tighter`

### C. Layout & Components
- **Glassmorphism Card:**
  ```tsx
  <div className="p-8 rounded-[2.5rem] bg-zinc-900/30 border border-zinc-800 backdrop-blur-md">
    {/* Content */}
  </div>
  ```
- **Grid System:** Responsive `grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6`

## 2. Coding Standards
### A. Component Structure
All components are located in `src/components` and adhere to the **Single Responsibility Principle (SRP)**.
- We prefer **Co-located Logic** (writing logic within the component) over extracting custom hooks for simple AI modifications.

### B. Data Fetching
- **Client Components:** Use the `useEffect` and `fetch('/api/...')` pattern.
- **Loading State:** Must verify data loading with **Skeleton UI** or a **Loading Spinner**.

### C. Type Definitions (TypeScript)
- Explicitly define Props and API response types using `interface`.
  ```typescript
  interface MarketData {
    ticker: string;
    price: number;
    change_pct: number;
  }
  ```

## 3. Vibe Coding Tips (AI Collaboration Guide)
Adding the following context to your prompt significantly improves frontend code quality:

> "Create a React component using Tailwind CSS. Use 'zinc' colors for a premium dark mode look. The design should feature glassmorphism, rounded corners (2.5rem), and sleek typography (Geist). Ensure it is fully responsive."
