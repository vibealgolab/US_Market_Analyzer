# Project Overview (v1.0)
**Date:** 2026-02-17
**Author:** VibeAlgoLab (AI Engineer)

## 1. Project Purpose
**US Market Analyzer** is a premium dashboard that visualizes vast US stock market data using AI (Gemini) and quantitative analysis tools, providing investors with intuitive insights.
The core mission of this project is **"transforming complex financial data into a beautiful and easy-to-understand UI."** To achieve this, we adopted the **Vibe Coding** methodology (high-speed development through AI collaboration).

## 3. Key Features & Pages
### A. Dashboard (Main)
- **Market Indicators:** Real-time status of major indices (S&P 500, Nasdaq, etc.)
- **AI Intelligence Report:** Daily market summary generated by Gemini (Rendered with RichText)
- **Market Sentiment:** Visualization of investor sentiment (Fear & Greed Index, etc.)

### B. Market Analysis (Detailed)
- **Sector Performance:** Heatmap and charts for 11 sector performances
- **ETF Flow:** Fund flow trends for major ETFs

### C. Portfolio Risk (Risk Management)
- **Correlation Heatmap:** Analysis of correlations between held assets
- **Volatility Analysis:** Diagnosis of portfolio volatility and risk exposure

## 4. Tech Stack
- **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS, Lucide Icons
- **Backend/Data:** Python 3.10+, Pandas, Google Gemini Pro API
- **Data Storage:** Local JSON/CSV (NoSQL-like structure)
- **Styling:** Design System (Zinc, Indigo, Rose Theme), Glassmorphism
